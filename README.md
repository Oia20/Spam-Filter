# Spam-Filter
This is a project that filters spam, it was a FCC project.
https://oia20.github.io/Spam-Filter/

What it filters:
- 'please help' and 'assist me'
- '[0-9] blank dollars'
- 'free money'
- 'stock alert'

Technologies used:
- HTML
- CSS
- Vanilla JS

  
Problems faced/concepts learned:
- this was great regex practice
